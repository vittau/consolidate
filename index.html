A system designed to get out of your way.

Built exclusively on markdown and standard folder structures.

Future-proof your memories

Proprietary databases lock your life behind subscriptions. Consolidate operates entirely on local `.md` files that you own forever.

The AI skills run locally on your machine via Claude Code, preserving the privacy of your most personal thoughts, while bringing enterprise-grade intelligence to your notebook.

The core philosophy is simple: writing should be instant and freeform. Organizing should be completely delegated to the machine. You write, it remembers.

Getting started

New to agentic AI? Here's the short version of what you need.

1

Install Claude Code

Consolidate runs on Claude Code, Anthropic's AI assistant for your terminal. It reads your notes, writes summaries, and keeps the journal organized on your behalf.

You'll need an Anthropic account and the Claude Code CLI installed. That's the only piece of AI you have to set up — everything else is plain files.

2

Create your journal

Run the install command once to scaffold a folder of markdown files. No database, no account to create, nothing proprietary — just a directory on your computer.

Then open that folder in Claude Code.

3

Write and consolidate

Start your day by running the /new-entry skill — it creates the markdown file for today so you have somewhere to write.

Jot down tasks, notes, and events using simple bullet notation. Whenever you want, run /consolidate and the agent rolls your entries into weekly and monthly reviews.

4

Ask your journal

Use the querying skills to ask questions in plain English — when something happened, what you focused on last month, how a project evolved.

The agent searches your notes and synthesizes an answer across months of context.
